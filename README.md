# parser_scrapy
Парсер информации о фильмах (scrapy, python) 

Парсер, который будет собирать информацию о фильмах:

Название, Жанр, Режиссер, Страна, Год

Результаты сохраняются в формате CSV.

Дополнительно: сбор рейтинга IMDB к уже собранным фильмам

Использована стартовая страница:

https://ru.wikipedia.org/wiki/%D0%9A%D0%B0%D1%82%D0%B5%D0%B3%D0%BE%D1%80%D0%B8%D1%8F:%D0%A4%D0%B8%D0%BB%D1%8C%D0%BC%D1%8B_%D0%BF%D0%BE_%D0%B0%D0%BB%D1%84%D0%B0%D0%B2%D0%B8%D1%82%D1%83

Запуск:

1) клонировать репозиторий и перейти в папку parser_scrapy

2) установить и активировать виртуальное окружение.

3) установить зависимости:
```
pip install -r requirements.txt
```
4) перерйти в папку spiders
```
cd scrapyproject/scrapyproject/spiders/
```
5) из папки spiders запустить парсер:
```
 scrapy crawl movies
 ```

 Пасрер можно остановить в любое время Ctrl+C

 Список фильмов сохраняется в файле movies.csv в папке spiders

 Список фильмов также сохранен в файле movies_done.csv
